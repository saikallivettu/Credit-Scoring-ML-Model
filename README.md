# Credit Scoring Classification Model

## Project Objective
An end-to-end Machine Learning model designed to predict consumer creditworthiness. The system analyzes financial behaviors to automatically categorize applicants into three distinct risk tiers: **Low Risk**, **Medium Risk**, or **High Risk**.

## Features Tracked
- Annual Income
- Existing Debts
- Length of Loan History
- Delinquent/Missed Payments
- Credit Utilization Ratio
- Current Account Balance

## Tech Stack & Frameworks
- **Language:** Python
- **Environment:** Jupyter Notebook / Google Colab
- **Libraries:** Pandas, NumPy, Scikit-Learn, Imbalanced-Learn (SMOTE), Seaborn, Matplotlib
- **Core Algorithm:** Random Forest Classifier

## Evaluation Performance
- **Accuracy:** ~90%+ 
- **Handling Class Imbalance:** SMOTE (Synthetic Minority Over-sampling Technique)
- **Metrics Calculated:** Precision, Recall, F1-Score, and Multi-class ROC-AUC
